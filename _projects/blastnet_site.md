**BLASTNet (Bearable Large Accessible Scientific Training Network-of-Datasets)** is a repository of high-fidelity reacting- and non-reacting fluid flow simulation data maintained by the [FX Lab](https://fxlab.stanford.edu).

I redesigned BLASTNet's interface with analytics synced from Kaggle API, built using a JSON cache with cron and the GitHub API along with Jekyll templating.

I also wrote detailed and extensive documentation to make it easy for other maintainers to add/modify data.
