blurb: >
  <p>
    <b>BLASTNet (Bearable Large Accessible Scientific Training Network-of-Datasets)</b> is a repository of high-fidelity reacting- and non-reacting fluid flow simulation data maintained by the <a href="https://fxlab.stanford.edu">FX Lab</a>.
  </p>
  <p>
    I redesigned BLASTNet's interface with analytics synced from Kaggle API, <br>
    built using a JSON cache with cron and the GitHub API along with Jekyll templating.
  </p>
  <p>
    I also wrote detailed and extensive documentation to make it easy for other maintainers to add/modify data.
  </p>
